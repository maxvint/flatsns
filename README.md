FlatSNS
=======

FlatSNS是移动优先的社区程序，基于Codeigniter和Twitter Bootstrap3。

##应用  

* 问答：社会化问答
* 图片：类Lofter、Tumblr的图片分享功能
* 论坛：轻论坛
* ……

##有什么不同？

####强调前端体验，例如：

* 美化的表单设计，参考：<http://www.bootcss.com/p/icheck>；

####性能、速度

* 优化速度，弃用模板引擎，大幅提升速度和性能；
* 分布式部署设计，数据库读写分离设计；
* 考虑内置coreseek搜索引擎，满足高精准的模糊搜索；
* 阿里云OSS、又拍云存储支持。

####运营支持

* 内置微信公众接口和运营工具；
* 积分消耗和充值（支付宝、财付通）功能；
                    

####功能模块可自定义

* 应用可根据需要开关，可以只开启一个或某几个模块。
